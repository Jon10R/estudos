Rproduzindo áudio

Vamos aprender a reprodusir áudio em HTML5.

O player acima foi adicionado usando a tag audio src="midia/audio.mp3"controls

A tag src= define o arquivo a ser reprodusido.

A tag controls adiciona o controlador do áudio no site.

Um exemplo do código:

        
    <audio> src="midia/audio.mp3"controls </audio>

Melhor comparibilidade com os navegadores

O exemplo acima pode não ser compatível com os navegadores, pois o seu formato é o mp3.

Outra maneira é a seguinte:

O exemplo acima possue o seguinte código:

        
            audio preload="metadata" controls loop>
                source src="midia/audio.mp3" type="audio/mpeg"
                source src="midia/audio.ogg" type="audio/ogg"
                source src="midia/audio.wav" type="audio/wav"
                p Infelizmente seu navegador não consegue reproduzir o áudio. 
                a href="midia/audio.mp3">Clique aqui para baixar o arquivo em mp3. /a /p
            /audio

A tag preload="metadata" define que ao abrir o site, apenas as informaçõs do arquivo serão caregadas, possibilitando uma maior fluidez para o site e economia de dados.

A tag controls exibe o controlador de áudio no site.

A tag loop define que quando o áudio chegar ao final ele reiniciará automaticamente.

O elemento source define o diretório do arquivo e o seu formato.

Podem ser adicionados varios formatos, porém se o navegador não tiver suporte para nenhum dos formatos, será exibida a menssagem que estiver no parágrafo seguinte.
