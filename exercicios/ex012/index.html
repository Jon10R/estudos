Inserindo videos.

Vídeos hospedados localmente.

Este vídeo está hospedado no meu próprio servidor.

Código:

            
                <video width="500" poster="imagens/imagen.jpg" controls>
                    <source src="midia/video.mp4" type="vide/mp4">
                    <source src="midia/video.webm" type="video/webm">
                    <p>O seu navegador não tem compatibilidade com reprodução de víde</p>
                </video>

O vídeo acima foi inserido com a tag video. Utilizando o parametro width=500, o vídeo foi apresentado com a largura de 500 pixels. Com o parametro poster, foi incluida a imagen do lago igapó como thumbnail. Com o parametro controls, foi adicionado o menu de reprodução. A tag source foi utilizada duas vezes, a primeira para indicar o vídeo no formato mp4, a segunda indica o formato webm. Logo a baixo, está a mensagen que será exibida caso nenhum dos dois formatos forém abertos.
